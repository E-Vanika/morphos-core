This folder holds the static gallery fallback images used by the web app when Supabase Storage is unavailable.

The active references live in `apps/web/app/page.tsx`:

- Craft: `pencil-art.jpeg`, `painting-landscape.jpeg`, `pencil-art-cryptography.jpeg`, and `painting-portrait.jpeg`
- Bridal: `bridal-bridesmaid-makeup.jpeg`, `bridal-saree-draping.jpeg`, `bridal-hairstyle.jpeg`, and `bridal-reception-hairstyle.jpeg`

Add permanent images to this folder, then add their exact filenames to the appropriate `FALLBACK_GALLERY_IMAGES` list in `page.tsx`.
